# hubot-random-dino

Post a random dino!

## Installation

In hubot project repo, run:

`npm install hubot-random-dino --save`

Then add **hubot-random-dino** to your `external-scripts.json`:

```json
[
  "hubot-random-dino"
]
```

## Sample Interaction

```
user1>> dino me
hubot>> <gif url>
user1>> raptor me
hubot>> <gif url>
```
